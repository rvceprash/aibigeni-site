  • Stripe Elements for secure payments

How it works

  1. User asks in plain language (voice or chat)
  2. CX extracts origin, destination, dates, cabin, pax
  3. Backend queries inventory & prices (Vertex AI Search / APIs)
  4. Gemini drafts natural confirmations or suggestions
  5. Stripe for payment → booking confirmation
  6. Firestore stores session & preferences securely
